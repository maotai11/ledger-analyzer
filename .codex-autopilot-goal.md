# Night Autopilot Goal (ledger-analyzer)

## Primary goal
Finish pending ledger-analyzer work end-to-end with verified results.

## Mandatory quality gates for every round
Before the final AUTOPILOT_STATUS block, output this block exactly:

[QUALITY_GATES]
closeout_done: yes|no
session_fresh: yes|no
artifact_controlled: yes|no
cache_checked: yes|no
branch_drift_checked: yes|no
verification_ran: yes|no
[/QUALITY_GATES]

Interpretation:
- closeout_done: all touched items are implemented and verified, not half-done.
- session_fresh: read latest local context/files before deciding status.
- artifact_controlled: build artifacts are in expected folders and no accidental duplicates.
- cache_checked: avoid unnecessary cache growth; clean temporary artifacts created in this round.
- branch_drift_checked: check for parallel-copy drift (if related copies exist, note impact).
- verification_ran: run at least one relevant verification command and report result.

## Execution policy
- Make real code or config progress each round.
- Keep changes minimal and reversible.
- Do not use destructive git commands.
- If blocked, provide exact blocker and smallest next action.

## Done definition
DONE only if all are true:
1. Implementation is complete for the scoped task.
2. Verification commands pass (or clearly documented limitation with evidence).
3. No unresolved high-impact TODOs introduced by this run.
4. AUTOPILOT_STATUS and QUALITY_GATES are both present.
